# FlavorLens : AI-Power Receipie Assistant

## Project Documentation

FlavorLens is a mobile app project that serves as an interactive, AI-powered recipe assistant. Our app provides personalized recipe suggestions using images, voice, or text as input. With a focus on meal planning and customization, FlavorLens uses generative AI, computer vision, and natural language processing (NLP) to enhance user experience.

## 1. Introduction

**FlavorLens** is being developed as a cross-platform Flutter mobile app. Our goal is to create an intuitive and visually appealing recipe assistant to simplify meal planning. By incorporating advanced AI technologies, the app can understand user input in various formats and suggest personalized recipe options based on preferences and dietary restrictions.

## 2. Project Overview

### What Does FlavorLens Do?

- **Input Methods:**
  - **Image:** Users can upload a picture of ingredients or a dish.
  - **Voice:** Users can speak their commands or queries.
  - **Text:** Users can type in their recipe requests or ingredient lists.
  
- **Output:**
  - Personalized recipe suggestions.
  - AI-generated creative recipe images to make meal planning more engaging.

- **Personalization:**
  - The app filters recipes based on dietary preferences like halal, vegetarian, vegan, etc.

### Technologies Used

- **Flutter:** The app is being developed using Flutter for cross-platform compatibility.
- **Generative AI & NLP:** We are integrating OpenAI's API for text generation and processing user queries.
- **Computer Vision:** The app uses image recognition technology to process food images uploaded by users.

## 3. Project Objectives

Our main objectives for **FlavorLens** include:

- **Integrate AI Components:**
  - Utilize APIs like OpenAI for text-based recipe suggestions, Whisper for voice-to-text, and DALL-E for AI-generated recipe images.
  
- **Provide Personalized Recipes:**
  - Offer recipe suggestions tailored to the user's specific dietary preferences, such as vegetarian, halal, gluten-free, etc.

- **Facilitate Meal Planning:**
  - Help users organize their meals with a shopping list feature and easy-to-follow meal planning tools.

- **Study Related Domains:**
  - Conduct research on similar apps, food trends, and meal planning systems to improve the app’s user experience.

## 4. Work Completed So Far

### AI Integration

- **OpenAI API:**
  - We are integrating OpenAI's API to generate recipe suggestions and instructions based on user input.
  
- **Whisper for Voice Input:**
  - The team has begun testing Whisper for converting voice commands into text, allowing the app to understand spoken instructions.

- **DALL-E for Image Generation:**
  - Early stages of implementing DALL-E to generate AI-enhanced images of suggested recipes.

### Personalization

- **Dietary Options:**
  - Development of a feature where users can set dietary preferences, and the app filters recipes accordingly (e.g., halal, vegetarian, etc.).

### Meal Planning Feature

- **To-Do List for Ingredients:**
  - Users can create shopping lists for the ingredients needed for their chosen recipes.
  
- **Meal Planning Layout:**
  - Meal planning features to help users organize their meals for the week with easy-to-follow instructions.

### Domain Analysis and Related Study

- **Research on Similar Apps:**
  - A comprehensive study of existing food and meal planning apps to understand trends and key features.

- **Study of Food and Nutrition:**
  - Analyzing food trends and nutrition needs to design a more useful app.

- **User Experience Research:**
  - Collecting user feedback from meal planning apps to improve user interaction and design.
